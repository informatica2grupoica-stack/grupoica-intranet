Resumen Ejecutivo

¿Qué es Comercial MP Workspace?

Es una intranet corporativa desarrollada a medida para Grupo ICA / Comercial MP SPA. Centraliza en una sola plataforma web las operaciones de inteligencia comercial, gestión de proveedores y clientes (integración con ERP Obuma), compras, contabilidad, RRHH, comunicación interna y herramientas de IA para análisis de precios de mercado.

La plataforma está desplegada en producción en Vercel (CD automático desde GitHub) con base de datos Supabase (PostgreSQL gestionado) y accesible desde cualquier dispositivo con navegador moderno.

URL de producción

https://grupoica-intranet.vercel.app · Autenticación via Supabase Auth (email + contraseña) · Sesiones persistentes con refresh token.

Módulos del Sistema

Descripción de Cada Módulo

MP-00

🏠 Inicio — Dashboard Principal

Página de bienvenida personalizada que muestra el nombre del usuario, fecha, una frase motivacional del día e indicadores económicos de Chile en tiempo real (USD, UF, UTM, EUR).

Contiene 5 widgets de resumen con links directos: mensajes nuevos en Chat, tareas pendientes, total de búsquedas en buscador, productos en historial de precios, y total de proveedores con calificación promedio.

Debajo de los widgets se muestran tres paneles compactos de acceso rápido: Tareas Pendientes (con gráfico de torta por prioridad), Chat Interno (últimos mensajes) y Mis Proveedores (gráfico de categorías). También aparece una alerta de productos sin stock del catálogo.

Funcionalidades clave

  • ✓ Indicadores económicos Chile (USD, UF, UTM, EUR) con variación vs. día anterior
  • ✓ Frase motivacional del día con botón "Inspiración" para renovar
  • ✓ KPIs en tiempo real conectados a Supabase y Obuma API
  • ✓ Alerta de stock crítico con link a buscador
  • ✓ Vista de nombre, rol y email del usuario logueado
MP-CHT

💬 Chat Interno

Sistema de mensajería privada entre colaboradores. Panel izquierdo con lista de colegas filtrable por nombre o cargo. Al seleccionar un colega se abre la conversación en panel derecho con historial completo de mensajes en tiempo real.

Funcionalidades clave

  • ✓ Mensajería en tiempo real (Supabase Realtime)
  • ✓ Buscador de colegas con filtro por nombre y cargo
  • ✓ Historial de mensajes persistente
  • ✓ Estado de conexión de usuarios
  • ✓ Privado y seguro (solo usuarios del sistema)
MP-TAR

✅ Gestión de Tareas

Módulo completo de gestión de tareas con 3 vistas intercambiables: Tabla, Gantt y Kanban. Permite crear, asignar, priorizar y dar seguimiento a tareas del equipo. Cada tarea tiene proyecto, responsable, prioridad (Alta/Media/Baja), fechas de inicio y vencimiento, estado (pendiente, en proceso, completada, atrasada) y comentarios.

Funcionalidades clave

  • ✓ Vista Tabla con columnas ordenables
  • ✓ Vista Gantt de cronograma temporal
  • ✓ Vista Kanban por estado
  • ✓ KPIs: total, completadas, en proceso, pendientes
  • ✓ Alerta visual de tareas atrasadas
  • ✓ Asignación de responsables por usuario del sistema
  • ✓ Comentarios y notas por tarea
MP-DSH

📊 Dashboard de Análisis

Panel de análisis en tiempo real que agrega datos del ERP Obuma en 4 KPIs principales: total de órdenes de compra (con monto total), clientes Obuma (activos vs. inactivos), movimientos de precio registrados, y alertas de stock. Incluye gráficos de evolución y distribución cargados bajo demanda.

Funcionalidades clave

  • ✓ KPIs en tiempo real desde API Obuma
  • ✓ Botón "Actualizar Todo" para forzar sincronización
  • ✓ Alertas de stock bajo y sin stock
  • ✓ Monto total de compras facturadas ($1.481.863.041)
  • ✓ Gráficos de distribución por categoría
MP-BUS

🔍 Monitor ICA — Buscador de Productos

Es el módulo principal de inteligencia de precios. Permite ingresar una lista de productos de forma manual o cargando un archivo Excel de costeo (formato COSTEO interno), buscar los precios actuales en 5 tiendas chilenas simultáneamente, y descargar el resultado en Excel con los precios encontrados.

Usa Serper API (Google Shopping) para obtener resultados de precios y Gemini 2.0 Flash para filtrar inteligentemente el producto más relevante entre los resultados. También ejecuta scrapers directos en Sodimac, Easy, Construmart, Imperial y MercadoLibre.

Los resultados se guardan automáticamente en el historial de precios de Supabase, permitiendo detectar variaciones en el tiempo.

Funcionalidades clave

  • ✓ Carga de Excel COSTEO (hasta 500+ productos por vez)
  • ✓ Carga de PDF de bases de licitación (extracción automática)
  • ✓ Lista manual de productos (texto libre)
  • ✓ Búsqueda concurrente en 5 tiendas
  • ✓ Filtro por región (Valparaíso, Metropolitana, Biobío, Aysén, Maule)
  • ✓ Gemini AI selecciona el mejor precio automáticamente
  • ✓ Exportar Excel con precios encontrados (formato COSTEO original)
  • ✓ Link directo a cada producto en tienda
  • ✓ Guardar búsquedas para reutilizar
MP-HIS

📈 Historial de Precios

Registro histórico de todos los precios encontrados por el buscador. Muestra tendencias (baja, alza, mantiene) con KPIs de ahorro potencial. Permite filtrar por tienda, tendencia, rango de fechas y buscar por nombre o SKU.

Funcionalidades clave

  • ✓ 682 registros históricos de precios
  • ✓ KPI de ahorro potencial ($588.355 en bajadas)
  • ✓ Filtros por tendencia, tienda, fecha
  • ✓ Indicadores visuales: BAJA (verde), ALZA (rojo), MANTIENE (gris)
  • ✓ Exportar a Excel
  • ✓ Link directo al producto en tienda

📍 Georeferencia

Mapa interactivo que localiza ferreterías, distribuidores de materiales de construcción, tiendas eléctricas, pinturas, maderas y otros proveedores locales en Chile. Usa Overpass API (OpenStreetMap) para consultar locales geolocalizados en tiempo real y los muestra en un mapa Leaflet.

Funcionalidades clave

  • ✓ Búsqueda por tipo de comercio (ferreterías, materiales, eléctrica, pinturas, maderas, plomería, mayorista, grandes tiendas)

  • ✓ Mapa interactivo con marcadores de locales
  • ✓ Datos en tiempo real desde OpenStreetMap
  • ✓ Acceso offline a resultados previos

Roles disponibles

Control de permisos por sección

Cada usuario (no admin) puede tener habilitadas/deshabilitadas secciones individuales del menú. El sistema redirige automáticamente a la primera sección permitida si accede a una URL bloqueada.
